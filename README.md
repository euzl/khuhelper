# Electric Car KHUhelper
전기차 충전소 운영과 사용을 위한 서비스

## 기능
- 증설 충전소 최적지 추천
- 현존 충전소 사용량 예측
- 열려있는 의사소통
- 실시간 충전소 현황 및 검색
